# Benchmark Cards

This directory contains benchmark cards for each built-in RLVR-Gym family.

Each card documents:

- the capability the family is intended to measure
- the formal task objective
- the verification guarantees available at step and trajectory level
- what oracle quality means
- recommended metrics to report
- a recommended in-distribution split protocol
- recommended out-of-distribution split protocols
- known shortcut risks
- baseline heuristics worth comparing against

Available cards:

- [graph_planning.md](graph_planning.md)
- [scheduling.md](scheduling.md)
- [symbolic_transformation.md](symbolic_transformation.md)
